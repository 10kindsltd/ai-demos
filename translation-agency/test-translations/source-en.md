# How real-time protection works

Warden Home watches every file the moment it's downloaded and checks it against known
threats. If something behaves like ransomware, we block it before it can run.

Real-time protection runs quietly in the background, with under 2% average CPU impact
during scans. You can start a manual scan any time from the dashboard, or let Warden
Home handle it automatically.

If Warden Home blocks something, you'll see a short notification explaining what
happened and what we did about it. No action is usually needed on your part.
