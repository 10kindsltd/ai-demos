# So funktioniert der Echtzeitschutz

Warden Home überprüft jede Datei in dem Moment, in dem sie heruntergeladen wird, und
gleicht sie mit bekannten Bedrohungen ab. Verhält sich eine Datei wie Ransomware,
blockieren wir sie, bevor sie ausgeführt werden kann.

Der Echtzeitschutz läuft unauffällig im Hintergrund, mit einer durchschnittlichen
CPU-Auslastung von unter 2 % während der Scans. Sie können jederzeit über das
Dashboard einen manuellen Scan starten oder Warden Home die Aufgabe automatisch
erledigen lassen.
