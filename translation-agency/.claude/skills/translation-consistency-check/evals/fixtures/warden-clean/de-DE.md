# Eine Ransomware-Attacke wurde blockiert

Warden Home hat eine Datei erkannt, die sich wie Ransomware verhielt, und sie
blockiert, bevor sie ausgeführt werden konnte. Sie müssen nichts weiter unternehmen.

Sie können die Details jederzeit über das Dashboard einsehen. Der Echtzeitschutz
läuft weiterhin unauffällig im Hintergrund und überprüft neue Dateien bei ihrem
Eintreffen.
